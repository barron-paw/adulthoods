# 前端更新后必须重新构建并部署

商品列表、顺序、媒体文件（1–12 对应产品1–6）都在代码里改好了。  
**如果线上页面还是只显示 3 个产品、顺序不对或图片加载失败，说明浏览器用的是旧的前端，需要重新构建并部署。**

## 操作步骤

1. **本地构建**
   ```bash
   cd frontend
   npm run build
   ```

2. **把 `frontend/dist` 里的全部文件上传到网站根目录**（覆盖原来的静态文件）。
   - 若用 Nginx 托管静态：把 dist 内容拷到 Nginx 的 root 目录。
   - 若用对象存储/CDN：把 dist 内容上传并覆盖。

3. **清缓存 / 强制刷新**
   - 浏览器：Ctrl+Shift+R（Windows）或 Cmd+Shift+R（Mac）。
   - 若用了 CDN（如 Cloudflare）：可在控制台里“清除缓存”或等缓存过期。
